# Agent 架构变更日志

## 加载链（上下游）

**上游（由谁加载本文件）：**
权威源。日常写入与周复盘时都可以追加。重大变更即时追加，不必等周复盘。

**同级联动：**
- `长期记忆.md` — 当前处境 + 跨周工作节点 + 周工作复盘走长期记忆，ITERATION_LOG 不记工作进展。
- `MEMORY/MEMORY_LOG.md` — MEMORY_LOG 只管记忆系统代谢，ITERATION_LOG 管所有架构级变更。两者互不重复。

---

## 关于此文件

**迭代日志**追踪 Agent 系统自身的有意义变更：系统架构、skill、hook、协议、区域 agent、身份层结构、文档大版本更新等。

不记录每日工作进展、跨周工作节点、记忆代谢条目、记忆周复盘或一般项目研究笔记。

---

## 变更类型

| 类型 | 说明 |
|---|---|
| 架构 | 系统结构变更 |
| 技能+ | 新技能创建 |
| 技能~ | 技能更新或废弃 |
| 优化 | 性能或体验改进 |
| 文档 | 文档更新（无代码变更） |
| 集成 | 外部工具连接 |

---

## 条目格式

条目格式权威源 = `storage-agent.md §ITERATION_LOG.md 一节`（三段式：摘要 / 改动 / 影响表；禁止字段与行数上限同源）。本文件不维护格式模板副本（v5.4 起防双源发散）。

---

版本真相以 `## 变更记录` 末条为准；头部不维护版本快照（防归档僵化）。
