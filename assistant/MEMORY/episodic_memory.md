## 文件职责

本文件保存 1-3 星的情景级候选 schema。它是程序型候选和语义型候选的统一池，用 `类型` 字段区分，不启动注入。

条目必须能写成可复现的"情境 -> 行动 / 预期"。只发生过的事实不进入本文件。

**容量阈值 / L0→L1 升格红线**：权威源 = `00.memory_agent.md §三池容量阈值 / §L0→L1 升格抽象红线`（v5.4 起本文件不复制规则本体）。
